Auto-generated Ruby gem providing 10 typed Datadog resource functions for the
Pangea infrastructure DSL. Generated by pangea-forge from
`datadog-terraform-resources` TOML specs.

## Build

```bash
bundle install
bundle exec rspec   # unit tests
bundle exec rake    # default task
```

## Architecture

```
lib/
  pangea-datadog.rb              # Gem entry point
  pangea-datadog/
    version.rb                   # PangeaDatadog::VERSION
  pangea/
    resources/
      datadog.rb                 # Provider registration
      datadog_monitor/           # DatadogMonitor resource
      datadog_dashboard/         # DatadogDashboard resource
      datadog_dashboard_json/    # DatadogDashboardJson resource
      datadog_service_level_objective/  # DatadogServiceLevelObjective
      datadog_synthetics_test/   # DatadogSyntheticsTest
      datadog_logs_index/        # DatadogLogsIndex
      datadog_logs_metric/       # DatadogLogsMetric
      datadog_logs_pipeline/     # DatadogLogsPipeline
      datadog_apm_retention_filter/    # DatadogApmRetentionFilter
      datadog_integration_aws/   # DatadogIntegrationAws
    types/
      datadog_types.rb           # Dry::Types module for Datadog
```

## Key Types

Each resource is a `Dry::Struct` subclass with typed attributes matching the
TOML spec. Used as: `Pangea::Resources::Datadog.datadog_monitor(name: ..., ...)`.
